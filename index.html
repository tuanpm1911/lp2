Prompt · Sẵn sàng copy
Điền thông tin ở trên — prompt tự tạo ở đây...
🤖 Mở Claude.ai 🚀 LadiPage
1
Nhấn Copy Prompt ở trên
2
Mở Claude.ai → tạo cuộc trò chuyện mới → paste prompt → nếu có upload ảnh thì đính kèm ảnh kèm theo → gửi
3
Claude trả về code HTML hoàn chỉnh — copy toàn bộ code đó (từ <!DOCTYPE> đến </html>)
4
Vào LadiPage → Tạo trang mới → Import HTML → paste code → kéo thả chỉnh sửa nội dung, ảnh, màu sắc
5
Thay ảnh placeholder bằng ảnh thực → chỉnh màu, font → Publish 🎉
